Contextual status message placed inline within a flow.

```jsx
<InlineNotification kind="success" title="Deployed" subtitle="Your app is live in us-south." />
<InlineNotification kind="error" title="Build failed" subtitle="See the logs for details." />
```

- `kind`: `error` / `success` / `warning` / `info`. Each sets the left-rule color, tint and icon.
